O que são Tags

Tags são comandos utilizados para informar o navegador como formatar um texto.

Primeiras Tags aprendidas

Title

A Tag title dentro do head modifica o nome que aparece na aba do navegador

h1, h2, h3, h4, h5 e h6

As Tags h1, h2, h3, h4, h5 e h6 são níveis de título e subtitulo variando o tamanho da letra sendo h1 a maior e h6 a menor

p

A Tag p gera um nível de texto do tipo parágrafo.

ol

A Tag ol gera lista de itens ordenados.

ul

A Tag ul gera lista de itens não ordenados.

i

A Tag i Configura a fonte do texto para itálico.

u

A Tag u configura o texto para ficar sublinhado.

strong

A Tag strong configura o texto para ficar negrito.

small

A Tag p configura o texto para ficar em tamanho pequeno.

a

A Tag a é utilizada para inserir um hyperlink no texto digitado.

img

A Tag img adiciona uma imagem à página.

input

A Tag input é utilizada para adicionar um campo para o usuário digitar.

Atributos de Tags

Tag a

Tag p

Na Tag a é preciso acrescentar os seguintes atributos:

  • href="URL" -> Referencia o link a ser redirecionado ao clicar no texto.
  • target= "blank ou self" -> Indica se ao clicar abrirá um nova pagina com o link (blank) ou se ira sobrepor a página que está sendo utilizada (self).
  • texto -> É o texto que aparece na página que ao ser clicado redireciona o usuário ao link configurado.
  • title="Nome" -> Nome que aparece quando passa o mouse em cima do texto sem clicar.

Tag img

Na Tag img é preciso acrescentar os seguintes atributos:

  • width="tamanho" -> Define o tamanho que a imagem aparecerá na página.
  • src= "local ou link da imagem" -> Indica o link de onde a imagem será puxada, pode ser local ou atraves de um link de site.

Tag input

Na Tag input é preciso acrescentar o seguintes atributo:

  • type="text ou number" -> determina o tipo de entrada, se for text a entrada é texto, se for number a entrada deve ser necessariamente um número.
